# Movie Booking Frontend

Frontend ini dibangun menggunakan **Next.js (App Router)**, **React**, dan **Tailwind CSS**.  
Fungsinya untuk menampilkan halaman movie, memilih seat, melakukan booking, dan melihat history.

## Fitur

### User
- Menampilkan daftar film
- Halaman detail film dengan pilihan showtime
- Memilih kursi pada seat selection page
- Payment page dengan konfirmasi booking
- User history page menampilkan tiket dan detail booking

### Admin
- Menambah, mengedit, menghapus film
- Menambahkan jam tayang ke film

## Instalasi

1. Clone repo frontend:
```bash
git clone <URL_FRONTEND_REPO>
cd <nama-folder-frontend>
```

2. Install dependencies:
```bash
npm install
```

3. Jalankan Development Server:
```bash
npm run dev
```

akses di browser: http://localhost:3000
